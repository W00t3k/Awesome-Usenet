# Majic Movie Selector

Small Flask app that recommends movies from a curated catalog using a natural-language prompt.

## Quick Start

1. Create and activate a virtual environment.
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run:
   `python -m app.main`
4. Open:
   `http://127.0.0.1:5001`

## Example Prompts

- `Need a funny sci-fi movie under 110 minutes`
- `Family movie night, something uplifting`
- `Recent drama that feels inspiring`

## Test

`pytest -q`
